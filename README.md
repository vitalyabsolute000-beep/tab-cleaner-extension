# tab-cleaner-extension
ENG:
This extension allows you to stop worrying about closing your favorite tabs.
It's now ready to show them all, leave only the selected ones, or delete only the selected ones.
You can also lock the tabs you need, which will prevent you from closing them with one click, only after confirmation.
RU:
Расширение позволяющее вам не беспокоиться за закрытие ваших нужных вкладок,
Теперь это расширение готово вам их все показать, оставить только выделенные или наоборот, удалить только выделенные,
Вы так же можете закрепить нужные вам вкладки в замочке которые не даст вам закрыть ее в один клик а только через подтверждение.
