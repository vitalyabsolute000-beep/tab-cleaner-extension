\# Tab Cleaner



A Chrome extension that helps you manage your open tabs —

it asks you which ones to keep and which to close, and lets you protect important tabs.



\## Features

\- View all open tabs with favicon

\- Bulk close marked tabs

\- "Keep only marked" mode

\- Protect important tabs from accidental closing



\## Installation

1\. Download or clone the repository

2\. Open chrome://extensions

3\. Enable "Developer mode"

4\. Click "Load unpacked extension" and specify the project folder



# Tab Cleaner



Расширение для Chrome, которое помогает разобраться с открытыми вкладками — 

спрашивает, какие оставить, а какие закрыть, и позволяет защищать важные вкладки.



\## Возможности

\- Просмотр всех открытых вкладок с favicon

\- Массовое закрытие отмеченных вкладок

\- Режим "оставить только отмеченные"

\- Защита важных вкладок от случайного закрытия



\## Установка

1\. Скачай или клонируй репозиторий

2\. Открой chrome://extensions

3\. Включи "Режим разработчика"

4\. Нажми "Загрузить распакованное расширение" и укажи папку проекта
